# Java Practice

This repository is where I practice Java fundamentals, problem solving, OOP concepts, collections, and interview-style coding patterns.

## Goals

- Refresh core Java knowledge
- Strengthen interview problem solving
- Practice backend-relevant Java concepts
- Build visible GitHub proof of continued learning

## Structure

- `basics/` - syntax, variables, control flow, foundational Java concepts
- `collections/` - HashMap, HashSet, List, Queue, Stack, and related patterns
- `problems/` - interview-style coding problems and solutions
- `oop/` - inheritance, polymorphism, abstraction, encapsulation examples
